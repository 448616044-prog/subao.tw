# 国际物流项目 — 长期记忆

## 项目背景
- 面向中国大陆市场的新项目
- 目标客户：外贸工厂、贸易公司、跨境电商卖家
- 业务方向：国际物流（中国→全球）
- 初步考虑的目的地：美国、东南亚、欧盟、日本等
- 大陆国际物流竞争格局：顺丰国际、4PX、云途、燕文 + 数百淘宝店，DR 60+ 大站林立

## 策略决策（2026-06-13 讨论结论）

### 域名策略：一个大站，按目的地分目录
```
主站（待定域名）/
├── /usa/        ← 寄美国专区
├── /japan/      ← 寄日本专区
├── /southeast-asia/ ← 东南亚专区
├── /eu/         ← 欧盟专区
├── /tools/      ← 运费计算器、关税查询
└── /blog/       ← 行业趋势、案例
```

理由：
- DR集中火力（多站分散=每个DR=0，一个大站统一积累）
- 内部链接网络效应
- 用户自然需要跨区域比较
- 品牌积累

❌ 拒绝策略：多站分区域（每个站从零建DR，维护成本3-5倍）

### 定位策略：不做又一個「国际物流平台」，做「外贸业务员的寄样百科」

❌ 红海不做：正面竞争"国际快递哪家好"（vs DHL DR90+/百运网 DR50+）
✅ 蓝海切入：超具体的外贸场景长尾词（竞品没有做SEO内容）

## 市场调研结果

### 已确认的两个核心缝隙

#### 🥇 缝隙一：外贸样品+寄样+敏感品类（立即可做）

需求验证：
- DHL 2026年2月专门发布「B2B 企业寄样品到海外」专题页 → 证明有搜索需求
- 百运网持续产出相关文章
- 但没有任何一家物流公司做了系统性的外贸样品SEO内容

用户画像：
- 外贸业务员搜索「发样品到德国客户，用DHL还是UPS？」
- 工厂跟单搜索「寄电子样品到美国要不要CE认证？」
- 这些长尾问题在知乎/外贸论坛被大量问，但从未被物流公司SEO承接

竞争状态：
- 第一页是搜狐通用文章、知乎通用文章、DHL品牌页
- 没有专业物流公司的SEO内容

#### 🥈 缝隙二：退运返修物流（先验证能力再干）

需求验证：
- 知乎 2024-2026年持续产出退运返修专题文章
- 搜狐 2026年5月发布退运返修全流程攻略
- 百度爱采购有活跃的产品列表（¥100-1500/票真实报价）
- 11467黄页多个公司（洋川、浩通、卓鹰、尼恩）持续更新
- Amazon FBA退货退回中国有专门需求

SERP分析：
- 搜索「退运返修怎么操作」→ TOP10 没有一家物流公司官网
- 搜索「退运返修 深圳 保税区 价格」→ 全是百度爱采购/11467 B2B广告
- 做退运返修的公司存在，但靠B2B平台付费广告获客，没有投SEO内容
- 第一个做退运返修SEO内容的物流公司可以直接收割搜索需求

风险：操作门槛高（保税区、保证金、海关监管），需要实际业务能力支撑

#### 🥉 缝隙三：ATA单证/展会物流（备选）

需求验证：知乎+搜狐有系统文章，中申国贸等报关公司做了内容
竞争状态：物流公司缺席，但需求量可能较前两个缝隙小

### 其他调研过的缝隙
- 墨西哥/巴西专线：墨西哥已有3-4家竞争，巴西几乎真空
- 含危品寄样（MSDS/化学品）：有需求但极垂直
- COD代收货款+物流：偏B2C，与外贸B2B方向不完全吻合
- 个人物品/留学生：已被DHL/FedEx/顺丰覆盖，竞争激烈

## 下一步计划

1. 确定域名
2. 建站：首页 + /usa/ pillar page + 运费对比工具
3. 第一批20个长尾关键词内容（外贸寄样方向）
4. 跑通SEO获客模型后 → 扩展退运返修
5. 两个缝隙闭环：「正常寄样→出问题退运」

## 关键原则
- 不要和DHL/百运网正面竞争泛词
- 做竞品没有做的超具体长尾词
- 先做门槛低的（外贸寄样），验证模型后再做高门槛的（退运返修）
- 内容策略：每一篇文章回答一个竞品没覆盖的具体问题
