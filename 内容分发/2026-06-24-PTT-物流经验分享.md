# PTT 發文：台灣寄大陸物流經驗分享

**看板**：e-shopping / 快遞物流
**標題**：[心得] 台灣寄大陸物流心得：泡麵/茶葉/化妝品一次搞定

**正文**：

之前都在潛水，今天來分享實際使用經驗。

家裡親戚在大陸工作，每個月固定寄一次東西過去。試過幾家物流後，簡單分享心得：

**寄過的品項**：
- 泡麵（維力炸醬麵素食版 → 可以寄）
- 茶葉（阿里山高山茶 → 可以寄）
- 化妝品（森田藥妝面膜 → 可以寄）
- 零食（乖乖/可樂果 → 可以寄）
- 滿漢大餐（含肉 → 可以寄但要走專線）

**各家比較**：
1. 郵局：最便宜但啥都不收（食品/液體/電池全拒）
2. 順豐：速度快但食品類拒收，液體也不行
3. 敏感貨專線：什麼都收，泡麵茶葉化妝品都OK，比順豐便宜

**費用參考**（我寄過的）：
- 食品類 NT$250/kg起
- 茶葉 NT$380/kg起
- 化妝品 NT$290/kg起

**結論**：如果你只是寄文件或衣服，郵局就好。但如果是食品/茶葉/化妝品這些敏感貨，一定要找能走專線的物流。

以上，有問題可以推文問，或自己搜LINE問物流業者。
