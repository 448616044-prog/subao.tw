## 发布步骤

### 一、搜狐号（最推荐先做）

1. 注册：https://mp.sohu.com
2. 实名认证（需要身份证）
3. 发布文章 → 选择「科技/互联网」或「生活」分类
4. 粘贴文章内容
5. 关键：文末作者简介保留超链接（搜狐号允许带链接）

### 二、网易号

1. 注册：https://mp.163.com
2. 需要提交身份信息审核（1-2个工作日）
3. 审核通过后发布文章
4. 分类选「生活」或「旅游」

### 三、知乎专栏

1. 登录知乎 → 创作中心 → 写文章
2. 粘贴内容
3. 添加标签：台湾、快递、物流、食品、保健品、茶叶
4. 文末链接知乎允许保留

## 三篇文章分配策略

**不要三篇发同一个平台。** 每个平台发不同的文章，最大化外链来源多样性：

| 平台 | 文章 | 理由 |
|------|------|------|
| 搜狐号 | 01-台湾美食寄大陆 | 美食类适合搜狐的大众读者 |
| 网易号 | 02-台湾保健品寄大陆 | 健康类适合网易的调性 |
| 知乎专栏 | 03-台湾茶叶寄大陆 | 茶叶话题在知乎有深度讨论氛围 |

## 发布时的注意事项

1. **标题可以微调**：各平台标题风格不同，搜狐偏标题党，知乎偏干货。可以根据平台调性微调
2. **不要堆关键词**：平台会检测SEO痕迹，自然写就好
3. **第一段最重要**：决定了推荐量，开头要抓人
4. **配图**：建议每篇文章配1-2张图（茶具、茶叶、保健品的实物图），增加推荐
5. **不要同时在三个平台发同一篇**：平台会判定为搬运，降权

## 外链策略

每篇文章文末都有作者简介，格式如下：

```
作者：速豹集运团队，专注台湾到大陆快递服务6年...
官网：https://subao.tw | 微信咨询：13026603164
```

这三篇文章发布后，你将获得：
- **3条高质量外链**（DR 80+ 域名）
- **品牌曝光**（三个平台月活过亿）
- **长期被动流量**（文章会持续被搜索和推荐）
