## 三大管道的真實對比

### 中華郵政（便宜但風險高）

| 項目 | 實際體驗 |
|:---|:---|
| 價格 | 普貨 NT$150-250/kg，最低消費 NT$200 左右 |
| 時效 | 5-15 天（不穩定） |
| 食品 | **不收敏感貨**（鳳梨酥、泡麵、茶葉等） |
| 化妝品 | 不收液體、粉末 |
| 退件率 | **高**，退件運費不退 |
| 追蹤 | 到海關後查不到狀態 |

適合：純普貨（衣服、書籍、日用品），不含任何敏感成分。

### 順豐快遞（快但限制多）

| 項目 | 實際體驗 |
|:---|:---|
| 價格 | 偏貴，小包裹起價 NT$200+ |
| 時效 | 3-7 天 |
| 食品 | **不收** |
| 化妝品 | 液體、粉末不收 |
| 茶葉 | **不收** |
| 保健品 | **不收** |
| 追蹤 | 即時查詢 |

適合：純普貨急件、文件、不含電池的3C（需確認）。

### 集運敏感貨專線（通關穩、品類最廣）

| 項目 | 實際體驗 |
|:---|:---|
| 價格 | NT$290/kg 起，最低消費 NT$290 |
| 時效 | 3-5 天 |
| 食品 | ✅ 鳳梨酥、泡麵、零食、茶葉 |
| 化妝品 | ✅ 面膜、精華液（24件以內） |
| 保健品 | ✅ 需先確認成分（動物源性不收） |
| 中藥 | ❌ 禁運 |
| 追蹤 | LINE 即時通知進度 |

適合：食品、保健品、化妝品、茶葉、含電池的3C、以及任何郵局不收的品類。**這是我現在唯一使用的管道。**

---

## 為什麼郵局最便宜但我不用了

最後一次用郵局，退件運費 + 重新寄送的雙重成本 + 等了兩週的時間成本——總體花費比集運直接寄貴了將近一倍。郵局的「便宜」只存在於順利通關的前提下，一旦退件就是血虧。

---

## 寄件前的 3 步檢查清單

1. **看成分**：食品看有沒有肉類成分，保健品看有沒有動物源性成分，化妝品看有沒有酒精/汞
2. **看數量**：同款超過 3 件、總數超過 24 件、重量超過 10kg——任何一條觸發都可能被認定為商業進口
3. **看申報**：誠實申報價值，800 元人民幣以下免稅
