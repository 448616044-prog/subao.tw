### 📋 问题分析
- **LCP 统一偏高**（3.2-4.0s vs 目标 <2.5s）：三页都是静态 HTML，延迟主要在图片加载和首屏渲染。

- **CLS 全部绿色**：布局稳定，无累积偏移问题。

### 🔧 优化建议（优先级排序）
1. **图片优化**：首页/文章页 LOGO 和首图使用 `<link rel="preload">` + 响应式 WebP
2. **Lazy loading**：文章页图片全部加 `loading="lazy"`
3. **CSS 关键路径**：提取首屏 CSS 内联到 `<head>`，延迟加载非关键 CSS
4. **Cloudflare 缓存**：确认 CF Pages 的缓存策略是否对 HTML 生效
