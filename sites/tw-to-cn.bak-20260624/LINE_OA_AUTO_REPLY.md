# LINE Official Account 自動回覆設定指南

以下內容適用於 LINE Official Account（LINE OA）管理後台：
👉 https://manager.line.biz/

---

## 🚀 自動回覆訊息設定

### 1. 加入好友歡迎訊息（首次互動觸發）

```
嗨！我是速豹 👋

感謝你加入速豹集運！

我是專營「台灣寄大陸」的跨境物流幫手。✨

你可以叫我幫手，以下是你最常用的问题，點一下就能得到答案：

🔹 【運費怎麼算？】
👉 回覆「1」

🔹 【什麼能寄？】
👉 回覆「2」

🔹 【多久到？】
👉 回覆「3」

🔹 【我要寄件！】
👉 直接告訴我：商品種類、數量、重量、寄到大陸哪個城市，我幫你估算！

💡 或者直接點下方按鈕估價：
👉 https://subao.tw/pricing-calculator
```

**設定方式**：LINE OA 後台 → 自動回覆 → 歡迎訊息 → 開啟 → 貼上以上內容

---

### 2. 關鍵字自動回覆設定

在 LINE OA 後台設定以下關鍵字觸發：

#### 關鍵字：「1」→ 運費說明
```
📦 速豹集運參考運費（台灣寄大陸）

以下為各類商品參考運費區間（每公斤，經濟實惠）：

🌟 A類｜藥品/奶粉/美妝/保健品
   華南地區：NT$320-480/kg
   華東地區：NT$340-590/kg
   華中/華北：NT$360-590/kg

🌟 B類｜液體/粉末保健品
   NT$410-700/kg

🌟 C類｜中西藥
   NT$370-1000/kg

🌟 D類｜電器
   NT$460-1370/kg

以上為經濟專線參考（海運，約兩週到）
空運快件專線約一週，費用較高。

✅ 精確報價：告訴我「商品種類＋重量＋目的地」，3分鐘內給你完整報價！

👉 想親自算？https://subao.tw/pricing-calculator
```

#### 關鍵字：「2」→ 什麼能寄
```
✅ 速豹可以幫你寄：

✔ 保健品（魚油、維他命、葉黃素等）
✔ 奶粉、燕窩
✔ 護膚品、化妝品（面膜、乳液等）
✔ 中藥材（人參、枸杞、當歸等，不含動物成分）
✔ 茶葉（工廠密封包裝）
✔ 食品（不含肉類，工廠包裝）
✔ 電器（注意海關核價）

❌ 無法寄送：
✗ 新鮮肉類、生雞蛋
✗ 含有動物成分的中藥（如鹿茸、熊膽、羚羊角）
✗ 槍械、毒品等違禁品

💡 不確定的商品，歡迎直接問我！
```

#### 關鍵字：「3」→ 多久到
```
⏱️ 速豹集運時效參考

🚢 經濟專線（海運）：約兩週送達
✈️ 快件專線（空運）：約一週送達

📍 目的地影響：
華南（福建/廣東）→ 最快
華東（上海/浙江）→ 其次
華中/華北 → 約一週
西南/東北/西北 → 時間較長

⚠️ 實際時效受海關通關速度影響，偏遠地區時間會更長。

💡 想確認你的商品寄到哪裡大約多久？告訴我目的地和商品種類，我幫你評估！
```

### 3. 忙碌時段自動回覆（保持專業感）

```
感謝訊息！我目前較忙碌，將於 30 分鐘內回覆你。

如果你需要快速獲得答案：
🔹 運費試算 👉 https://subao.tw/pricing-calculator
🔹 禁運品查詢 👉 https://subao.tw/blog/tw-to-cn-prohibited-items
🔹 寄件流程 👉 https://subao.tw/blog/tw-to-cn-shipping-guide

或繼續留言，我會盡快回覆你！🚚
```

**設定方式**：LINE OA 後台 → 自動回覆 → 營業時間設定 → 關閉時回覆 → 貼上以上內容

### 4. 主動標籤與分眾訊息（行銷模組）

在 LINE OA 後台設定「標籤」功能：

| 標籤 | 條件 | 發送內容 |
|:---|:---|:---|
| 新好友 | 首次加入 | 歡迎訊息（見上方） |
| 已詢價 | 互動過但未成交 | 3天後：「Amy您好，上次的面膜寄送想好了嗎？隨時可以幫你安排～」 |
| 已成交 | 確認寄件後 | 發貨通知＋到貨提醒 |
| 活躍用戶 | 多次互動 | 節慶關懷（中秋/春節優惠） |
