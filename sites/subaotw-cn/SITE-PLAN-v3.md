### Schema 规范

每个页面必须包含：
```
<script type="application/ld+json">
  { "@context":"https://schema.org", "@type":"Article", ... }
</script>
<script type="application/ld+json">
  { "@context":"https://schema.org", "@type":"BreadcrumbList", ... }
</script>
<script type="application/ld+json">
  { "@context":"https://schema.org", "@type":"FAQPage", ... }  // 有FAQ的页面
</script>
```

## 七、关键决策记录

| 决策 | 结论 | 原因 |
|------|------|------|
| 不做小件普货 | ✅ | 卡菲斯/飞晋垄断，竞争饱和 |
