# 速豹集運 LINE 客服轉化手冊

> 目標：報價 → 下單轉化率提升至 30%+

---

## 一、標準應對話術

### 場景 1：客戶第一次諮詢
```
客戶：「請問寄XX到大陸多少錢？」
```

**標準回覆**（30秒內回）：
> 「您好！請問是什麼東西？多重？寄到哪個城市？方便傳照片給我嗎？2分鐘內幫您報價 😊」

### 場景 2：報價後 — 這是轉化關鍵點 ⚠️

**❌ 錯誤做法**：報完價就等客戶回
**✅ 正確做法**：報價後立刻接促單

```
「這是今天的價格，運費每週會浮動調整。
您今天確認的話，我馬上安排，最快週三就能到○○。
需要幫您先保留這個價格嗎？」
```

### 場景 3：客戶說「好的謝謝，我再考慮」

**即時跟進**：
```
「沒問題！這個價格我幫您保留48小時（到○月○日晚上），
過期要重新報價哦。有任何問題隨時找我 😊」
```

### 場景 4：24小時後跟進
```
「哈囉～昨天報的價格還在有效期内。
有決定好要寄嗎？今天出貨的話週五前到哦 🚀」
```

### 場景 5：客戶說「別家比較便宜」

```
「明白！市場上確實有更便宜的。
不過提醒您確認三點：
1. 這個價格含關稅嗎？（很多報價不含，到了要補）
2. 走的是敏感貨專線嗎？（普貨通道寄食品會被退）
3. 有沒有包裝費/派送費？（有些結賬才加）
我們是全部包在報價裡，到了不用再補任何費用。」
```

### 場景 6：客戶下單後
```
「收到！您的包裹預計○月○日寄出，○月○日前送達。
出貨後會給您單號，隨時可以查。
有問題直接LINE我 👍」
```

## 二、客戶跟進節奏

| 時間點 | 動作 | 話術重點 |
|--------|------|----------|
| **報價後即時** | 促單+限時 | 「今天確認馬上安排，價格保留48小時」 |
| **24小時後** | 第一次跟進 | 「價格還在有效期，今天出貨週五到」 |
| **48小時後** | 第二次跟進（最後提醒） | 「價格今晚過期，明天報價可能有變動」 |
| **7天後** | 長尾喚醒 | 「最近有新優惠，寄10kg以上免派送費」 |
| **30天後** | 節日/促銷群發 | 「端午節前最後一波，現在寄節前到」 |

## 三、常見問題標準回答

### Q：會不會被海關扣？
> 「我們走敏感貨專線通道，不是一般快遞。食品/保健品/化妝品通關率 99% 以上，被扣全額退款。」

### Q：包裹會不見嗎？
> 「每件都有基本保險。萬一真的遺失，賠付最高運費三倍 + 貨值理賠。6年來沒有客戶因為包裹遺失受過損失。」

### Q：可以貨到付款嗎？
> 「可以！到大陸後支付，支援支付寶、微信、銀行轉帳。線上付款也可。」

### Q：能開收據/發票嗎？
> 「可以開電子收據。如需統一發票請提前告知。」
