## 一、现状诊断：关键词漏斗倒挂

```
           ┌──────────────────────┐
  HEAD     │ 台灣寄大陸           │ ← 排名 18.5，几乎无流量
  (大词)   │ 台灣寄大陸快遞       │ ← 未进前20
           │ 台灣寄大陸運費       │ ← 未进前20
           └──────────────────────┘
           ┌──────────────────────┐
  MID      │ 台灣零食寄大陸 (7.8) │ ← 排名 6-9，滴滴流量
  (品类词) │ 台灣伴手禮寄大陸(6.2)│
           │ 台灣寄東西到大陸(16) │
           └──────────────────────┘
           ┌──────────────────────┐ ← 🔥 当前流量主力
  LONG     │ 鳳梨酥可以寄大陸嗎   │   排名 4-8，占比80%+
  (问句)   │ 泡麵可以寄大陸嗎     │   但：搜索量小、天花板低
           │ 茶葉可以寄大陸嗎     │
           │ 面膜可以寄大陸嗎     │
           └──────────────────────┘
```

**核心矛盾**：在一个「倒挂漏斗」里，大词没排名 = 规模化增长被卡死。

**当前流量天花板估算**：
- 长尾问句 × 平均搜索量 ≈ 500-800 月搜索
- CTR 5-10% → 每月 ~50-80 次点击
- 品类词 × 2000-5000 月搜索 → **这才是真正的流量金矿**

### 路径详解

#### 路径 1：搜索多引擎（低垂果实）

| 引擎 | 当前流量 | 潜力 | 动作 |
|:---|:---:|:---:|:---|
| Google | 804 会话 | 主体 | 持续优化 |
| Yahoo TW | 101 会话 | ★★★★ | 提交 sitemap 到 Yahoo Site Explorer |
| Bing | 57 会话 | ★★★ | 提交到 Bing Webmaster Tools |
| ChatGPT | 20 会话 | ★★★★★ | 结构化数据优化（AI 引用首选 Schema 页面） |
| Perplexity | 未追踪 | ★★★★ | 同上，Schema 覆盖 |

**ChatGPT/Perplexity 优化是巨大的新机会**：
- 当前已从 chatgpt.com 获得 20 会话（纯自然引用，零优化投入）
- AI 搜索引擎优先引用有完整 Schema、结构化好的页面
- 速豹 45 篇 blog 全部已有 Article Schema → **天然优势**
- 动作：给 FAQ Schema 页面加 `@type: Question` + `acceptedAnswer` 让 AI 直接抓答案

#### 路径 2：内容社区分发（台湾本地用户触达）

| 平台 | 策略 | 预期获客/月 |
|:---|:---|:---:|
| **Dcard** | 在「网路购物」「生活」板发经验文「第一次寄大陆被税？」 | 50-100 会话 |
| **PTT** | 「快递板」「集运板」回答寄大陆问题，签名档放链接 | 20-50 |
| **Matters** | 发深度攻略文（已有账号），被 Google 收录后形成第二站群 | 30-80 |
| **Medium** | 繁体中文版同步发（已有账号），双收录 | 20-50 |
| **Pixnet** | 痞客邦部落格（台湾用户信任），长文版本同步 | 50-100 |

**关键：这些平台自己的站内搜索也是流量来源，不依赖 Google。**

#### 路径 3：LINE OA 私域（复购引擎）

- 当前 LINE 好友：估计 50-100 人（基于会话量估算）
- 每新增一个寄件客户 → 自动加入 LINE 好友
- 每月推送 1-2 次（节假日提醒、运费优惠、新路线开通）
- **复购率提升比拉新 ROI 高 5-10 倍**

#### 路径 4：品牌 PR（加速器）

- Google Business Profile 注册（台湾地址）
- 上 1 次台湾物流/电商媒体采访 → 外链 + 品牌搜索量暴增
- 参加台湾物流展会/跨境电商论坛 → 线下获客 + 品牌背书
- 台湾 YouTuber 开箱合作（寄一个包裹全程跟拍）

#### 路径 5：跨境市场拓展（规模化）

- 当 subao.tw 在台湾寄大陆领域站稳后
- 复制模式到日本、韩国、新加坡
- 每个市场独立域名 + 本地化内容
- 共享后端物流网络
