# Taiwan to China Shipping Calculator

Free instant shipping cost estimator for parcels from Taiwan to Mainland China. Supports general goods, sensitive goods (food/supplements/cosmetics), and special goods (electronics with batteries).

## Features

- Input weight and category → instant NT$ estimate
- Supports 3 categories: general / sensitive / special
- Regional pricing (East/South/North/Central/West China)
- Tax-inclusive pricing (包稅雙清)
- No dependencies, single HTML file

## Live Demo

Open `index.html` in any browser, or visit the [GitHub Pages demo](https://subao-logistics.github.io/tw-cn-shipping-calc/).
