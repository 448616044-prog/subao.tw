## 三、内容优化策略

### 3.1 应对AI Overview：从"被回答"到"被引用"

Google AI Overview会从排名靠前的页面抓取内容生成摘要。我们的目标不是阻止AI Overview，而是**让自己的内容被AI Overview引用**，并在摘要中突出品牌。

**具体做法**：

1. **每篇文章开头加一个"一句话结论"区块**
   ```html
   <div class="ai-summary-box">
     <strong>速豹结论：</strong>凤梨酥可以寄大陆，但必须是不含蛋黄/肉类的纯凤梨酥。
     推荐选择经济专线，NT$320/kg起，7-14个工作天送达。
     <a href="https://line.me/ti/p/~subaotw5988">免费评估你的凤梨酥能不能寄 →</a>
   </div>
   ```
   - 位置：H1下方、正文第一段之前
   - 格式：结构化、带数字、带结论、带品牌名
   - 目的：AI Overview抓取时大概率会引用这段，用户看到"速豹结论"就记住了品牌

2. **FAQ Schema必须全站部署**
   - AI Overview特别喜欢抓取FAQ格式的内容
   - 我们已经做了，但需要确保每个页面的FAQ都覆盖AI Overview已经回答的问题
   - 凤梨酥页面的FAQ必须包含："凤梨酥可以寄大陆吗？""蛋黄酥可以寄吗？""会被税吗？"

3. **内容深度超过AI Overview**
   - AI Overview只能给3-5条概括性答案
   - 我们的文章需要给出**具体可操作指南**（包装方法、申报价值、选择什么渠道）
   - 用户看了AI Overview后，如果想"具体怎么操作"，就会点进我们的深度文章

### 3.2 针对iCarry的内容碾压策略

用户标注"iCarry呈现内容不多"——这就是机会。

| iCarry的页面特征 | 我们的差异化 |
|:---|:---|
| 礼品电商导向，重产品轻攻略 | 专注物流攻略，解决"能不能寄+怎么寄" |
| 单页面内容量估计<800字 | 目标3000+字，覆盖全流程 |
| 无海关/税务实操细节 | 深度讲解申报、税金、退件处理 |
| 无运输渠道对比 | 做对比表格（经济/快递/敏感货专线） |

**具体行动**：
- food-shipping-guide：扩充至3000字，增加"2026年最新海关规定"章节
- tw-to-cn-prohibited-items：已经是3000字+，保持优势
- tea-shipping-guide：已扩充至4000字，继续保持领先
- 每篇文章增加"常见问题+速豹方案"板块，直接给出可执行的解决方案

### 3.3 抢占"零点击"关键词的变体

用户说"至少少了35%的点击流量"，是因为AI Overview覆盖了头部query。我们需要抢占**长尾变体**：

| AI Overview已覆盖 | 我们主攻的长尾变体 |
|:---|:---|
| 凤梨酥可以寄大陆吗 | "凤梨酥寄大陆会被税吗""佳德凤梨酥可以寄大陆吗""凤梨酥寄北京要几天" |
| 茶叶可以寄大陆吗 | "阿里山茶叶寄大陆规定""台湾高山茶寄大陆2026""茶叶寄大陆要申报吗" |
| 保健品可以寄大陆吗 | "鱼油寄大陆限量""维骨力可以寄大陆吗""胶原蛋白粉寄大陆规定" |

**操作方法**：
- 用GSC的Search Analytics导出所有query，筛选包含品牌名/具体地名的长尾词
- 针对这些词创建FAQ条目或独立段落
- 在文章中自然嵌入这些长尾query的完整问句

### 5.1 垃圾外链清理（紧急，本周必须做）

**问题外链来源**：
- `ranklinkpro.shop` — 外链贩卖平台（立刻disavow）
- `fiverr-seo-for-business-growth.site` — Fiverr外链服务（立刻disavow）

**执行步骤**：
1. 登录 Google Search Console → "链接" → "外部链接" → "热门链接来源"
2. 导出完整外链列表，筛选所有可疑域名
3. 创建 `disavow.txt`：

```
# disavow.txt — subao.tw 垃圾外链清理
# 创建日期：2026-05-26
domain:ranklinkpro.shop
domain:fiverr-seo-for-business-growth.site
```

4. 提交地址：Google Search Console → 法规提醒 → "链接评估工具" → 上传disavow.txt

**注意**：disavow是告诉Google"忽略这些链接"，不是删除。已存在的垃圾外链不影响，但提交后Google不会再用它们评判你的网站。

## 八、总结

### 核心结论

1. **外链是当前最大瓶颈**：DR=0意味着Google不信任这个域名，再优质的内容也难以排名。外链建设必须是接下来6个月的最高优先级。

2. **iCarry的模式可以直接复制**：它的外链不是买的，是商业合作自然产生的。我们作为物流服务商，完全可以和台湾品牌谈互链合作。

3. **内容是杠杆，但外链是支点**：今天做的内容扩充、CTA优化都是在"打磨杠杆"，但没有外链这个"支点"，杠杆撬不动排名。

4. **AI Overview不是敌人，是机会**：被AI Overview引用反而能增加品牌曝光。我们需要优化内容结构，让自己成为AI Overview的信息来源。

5. **Facebook/社交是低成本突破口**：在独立站DR还低的时候，社交媒体的内容可以帮我们获取初始流量和信任。

### 一句话战略

> **用商业合作获取真实外链（学iCarry），用内容深度碾压竞品（打iCarry软肋），用社交内容补充流量（借Facebook权重），用结构化内容攻占AI Overview（抢零点击时代的品牌曝光）。**
