## 四、DR（Domain Rating）分析

### 4.1 当前 DR 数据

| 域名 | DR | 评级 | 数据源 |
|:---|:---:|:---|:---|
| **subao.tw** | **2.0** | 🔴 极低 | Ahrefs API (实时) |
| subaotw.cn | 0.0 | 🔴 零 | Ahrefs API (实时) |

> DR 2.0 在 Ahrefs 的 0-100 对数刻度上属于「非常弱的网站」。对比：一般能稳定获取搜索流量的站点 DR 通常在 10-30。

### 4.2 DR 对排名的影响（理论）

根据 Ahrefs 的研究，DR 与搜索流量有显著正相关。DR 2.0 意味着：
- 几乎没有来自高质量域名的外链
- Google 认为这个站点的"权威度"很低
- 竞争中等难度以上关键词时处于劣势

### 4.3 当前 DR 停滞原因（7/4 DR=2 → 7/18 DR=2）

| 可能原因 | 说明 |
|:---|:---|
| 外链建设未执行 | 28个外链目标方案已出，但用户尚未手动提交 |
| Ahrefs 爬虫抓取延迟 | 免费 DR API 的数据更新频率较低，可能已有新外链但未反映 |
| 现有外链质量低 | MEMORY 中记录的 276 个引荐域名可能包含大量低质量/spam域名 |
| 站点年龄不足 | subao.tw 上线不到2个月，天然处于沙盒期 |

### 4.4 DR 提升路径

```
当前 DR=2 → 目标 DR=10 (3个月) → 目标 DR=20 (6个月)
```

| 阶段 | 动作 | 预期 DR | 时间线 |
|:---:|:---|:---:|:---|
| 立即 | 提交第一梯队 8 个外链（Ruzave/Manta/Crunchbase/About.me 等） | 2→5 | 1-2周 |
| 短期 | 内容分发到 Pixnet/Medium/Matters/Dcard | 5→8 | 1个月 |
| 中期 | 数字 PR + 行业目录 + 合作博客 | 8→15 | 3个月 |
| 长期 | 原创数据/工具吸引自然外链 | 15→20+ | 6个月 |

### 4.5 DR 低 ≠ 不能排名

**重要**：DR 是第三方指标，Google 不使用 DR。当前 subao.tw 在 DR=2 的情况下已经：
- 63 个关键词进入 Google Top 20
- 多个问句式长尾词排 P3-P5
- 日均 60-80 点击且持续增长

这说明 **内容相关性 > DR**，在台湾→大陆物流这个细分领域竞争度低，即使 DR 低也能靠内容质量取胜。
