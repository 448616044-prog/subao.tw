## 二、内容标准模板

每篇文章统一格式：

```
標題（H1）：精準關鍵詞 + 吸引力標題
Meta描述：150字內，包含關鍵詞+CTA

內容結構：
1. 前言鉤子（1段，引發共鳴）
2. 目錄/概述
3. 正文（分點說明，每點2-4段）
4. 常見問題FAQ（5個問題）
5. CTA區塊（LINE/聯繫方式）

技術SEO：
- H1：包含核心關鍵詞
- H2：長尾關鍵詞
- 內鏈：連結到網站其他相關文章
- 圖片：壓縮+ALT標籤（描述性繁體文字）
- canonical URL
```
