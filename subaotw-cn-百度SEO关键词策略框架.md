| 搜狐/163自媒体 | 内容分发 | 资讯型关键词 | 不能直接转化，引流弱 |

**结论**：B2B 大件物流领域竞争者内容薄弱，百度知道/知乎占据大量长尾词，正式企业站内容质量普遍偏低 — 这是 subaotw.cn 的机会窗口。

## 四、视频策略

### 视频为什么要做？
- 百度搜索结果中视频结果占比 15-25%
- 百度视频源：好看视频（百度自家）、B站、腾讯视频
- 视频自动播放缩略图在 SERP 中点击率高 30%+
- 百度小程序支持视频嵌入

### 视频放哪里用户体验最好？

| 位置 | 适合视频类型 | 优先级 |
|------|------------|--------|
| **首页 Hero 下方** | 30秒品牌介绍「速豹一镜到底看仓库/装车/报关」 | 🔴 P0 |
| **设备品类页侧栏** | 该品类实拍「CNC机床包装过程/装柜」 | 🔴 P0 |
| **案例详情页顶部** | 客户案例纪录片「东莞→基隆 8吨设备全程」 | 🟡 P1 |
| **物流指南 Pillar 页** | 教育型「设备出口报关5步走」「木箱包装演示」 | 🟡 P1 |
| **运费计算器旁** | 引导型「30秒教你怎么算运费」 | 🟢 P2 |
| **关于我们页** | 信任型「仓库实拍+团队介绍+客户证言」 | 🟢 P2 |

### 视频嵌入方案

**方案 A：好看视频（推荐）**
- 百度自家平台，视频在百度搜索中优先展示
- 上传到好看视频 → 取嵌入代码 → 放到页面中
- 优点：百度搜索加权、国内访问快
- 缺点：需要注册好看视频创作者账号

**方案 B：B站嵌入**
- B站视频在百度搜索结果中有露出
- 嵌入代码简单：`<iframe src="//player.bilibili.com/...">`
- 优点：用户基础大、弹幕互动
- 缺点：有B站水印、可能有广告

**方案 C：自建视频+Schema 标记**
- 视频直接放服务器，用 VideoObject Schema
- 优点：完全可控、无第三方水印
- 缺点：服务器带宽消耗、无百度视频源入口

**推荐组合**：好看视频（上传）+ B站（分发）+ 页面 VideoObject Schema
